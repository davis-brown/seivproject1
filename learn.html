Learn More

What can I do to learn more about Computer Science?


There are numerous resources that can be utilized to learn more about computer science. Computer science is a very broad field, so the resources provided are geared towards programming. They do not include all asplects of Computer Science. This short list should help you start your journey or at least point you in the right direction.


How to Learn More


  • Codecademy


    • Codecademy provides a self-driven, interactive learning experience for the top programming languages in the industry. It offers a free tier, which is sufficient for anyone at the beginner level.
    Codecademy also offers a paid tier that offers personal learning paths and many other benefits.


  • W3Schools


    • W3Schools offers a vast collection of web development resources in tutorials and examples. This collection specifically includes AJAX, SQL, ASP, CSS, JavaScript and HTML markup code.


  • Code with Google


    • Code with Google is another self-driven learning experience that accomodates aspiring computer scientists of all ages. This site aims to make the process of learning programming fun and interactive.


  • Github


    • Github is the largest collection of open source code examples. It also provides beginners a way to connect with other beginners and experienced computer science professionals.
